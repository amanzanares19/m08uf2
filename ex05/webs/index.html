Introducció NodeJs

Asíncron i controlat per esdeveniments: totes les API de la biblioteca Node.js són asíncrones,
és a dir, sensebloqueig.

Molt ràpid: en estar construït al motor JavaScript V8 de Google Chrome,
la biblioteca Node.js és molt ràpida en l'execució de codi.

Processos en un sol fil però molt escalable: Node.js utilitza un model d'un sol fil amb bucle d'esdeveniments. El mecanisme d'esdeveniments ajuda el servidor a respondre sense bloquejos i fa que el servidor sigui altament escalable en comparació
dels servidors tradicionals que creen fils limitats per gestionar les sol·licituds. Node.js utilitza un sol programa de subprocessos i el mateix programa pot proporcionar servei a un nombre molt més gran
de sol·licituds que els servidors tradicionals com el servidor Apache HTTP.
